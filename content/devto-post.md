# 🚀 Build a Production AI Agent in 5 Minutes

## The Problem

Every developer wants to build an AI agent. But starting from scratch takes:
- Days to set up the framework
- Hours to integrate providers
- More time for MCP tools

## The Solution

Introducing **Agent Starter** - a production-ready template that just works.

### Features
- 🔄 Multi-provider (OpenAI, Anthropic, Ollama)
- 🔌 MCP integration built-in  
- 🌐 Web UI included
- 🚀 One-click deploy

### Quick Start

```bash
git clone https://github.com/RedKnight-aj/agent-starter.git
cd agent-starter
npm install
cp .env.example .env
npm run dev
```

That's it! 5 minutes max.

## Why It Works

1. **Proven framework** - Not starting from scratch
2. **Production-ready** - Security, rate limiting included
3. **Extensible** - Easy to add more providers
