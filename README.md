# Agent Starter - Production-Ready AI Agent Template

[![Stars](https://img.shields.io/github/stars/RedKnight-aj/agent-starter?style=flat)](https://github.com/RedKnight-aj/agent-starter)

Deploy a production-ready AI agent in 5 minutes.

## Features
- Multi-provider: OpenAI, Anthropic, Ollama
- MCP Integration
- Web UI included
- One-click deploy

## Quick Start
```bash
git clone https://github.com/RedKnight-aj/agent-starter.git
cd agent-starter
cp .env.example .env
# Add your API keys
npm install
npm run dev
```

## Deploy
- Vercel: One-click
- Railway: One-click
- Docker: `docker-compose up`
